## Contributing

Contributions are welcome and encouraged! Whether you're fixing a bug, improving documentation, or adding a new feature, feel free to get involved.

### How to contribute

1. **Fork** the repository
2. **Create a branch** for your changes:
   ```bash
   git checkout -b fix/your-bug-description
   ```
3. **Make your changes** and commit them:
   ```bash
   git commit -m "Fix: description of what you changed"
   ```
4. **Push** to your fork:
   ```bash
   git push origin fix/your-bug-description
   ```
5. **Open a Pull Request** against the `main` branch

### Guidelines

- Please keep pull requests focused — one fix or feature per PR
- Write clear commit messages describing what changed and why
- If you're fixing a bug, consider adding a note about how to reproduce it
- Be respectful and constructive in code reviews

### Reporting Issues

Found a bug or have a feature request? [Open an issue](https://github.com/your-username/infuse-plugin/issues) and describe the problem or suggestion clearly.
